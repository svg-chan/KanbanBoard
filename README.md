# KanbanBoard
A minimal Kanban board for people who don't want to sign up for another SaaS. Four columns (To Do, In Progress, Review, Done), drag-and-drop, task priorities, due dates, and review notes. Nothing else.

One Board
Finish what you started → Archive it → Begin again with an empty mind.

100% Local. All data stays in your browser's storage.
No tracking, no ads, no analytics, no cookies
Export/Import anytime as JSON. Delete anytime by clearing browser data.
Works offline. Just download open source files and run it yourself.
